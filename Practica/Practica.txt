Pasos git
1- Creamos una carpeta: mkdir"Nombre de la carpeta"
2- Creamos la rama <master> : git init
3- Agregamos nuestros datos: git config --local user.name"Agregamos nuestro nombre"
   			     git config --local user.email"Agregamos nuestro correo"
4- Checkeamos la informacion: git config -list
5- Para borrar alguna Carpeta o Archivo: Rdmir"nombre de la carpeta o archivo" 
6- Para Agregar algun archivo al la zona de preparacion: git add "Nombre de el archivo" / git add . (Agg todo los archivos)
7- Para ver el estado de el archivo: git status
8- Como commitear archivos: git commit (con "a" agregamos un texto, "esc" y despues ponemos ":" y escribimos wq!)
                           <git commit -a "Agregamos un comentario"
